# CRUD Personas en Python

Proyecto básico de CRUD (Create, Read, Update, Delete) en Python usando:
- dataclasses
- persistencia en archivos JSON
- menú por consola

## Funcionalidades
- Crear personas
- Listar personas
- Modificar personas
- Eliminar personas
- Guardado automático al salir

## Requisitos
- Python 3.10+

## Uso
Ejecutar:
```bash
python main.py
